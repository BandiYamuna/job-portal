# Job Portal Application

This is a full-stack Job Portal web application developed using Spring Boot and Thymeleaf. The application allows users to register, login, and apply for jobs, while admins can manage job postings.

## 🚀 Technologies Used

- Java
- Spring Boot
- Spring MVC
- Thymeleaf
- MySQL
- HTML, CSS
- Maven

## 📌 Features

- User Registration and Login
- Job Listings
- Apply for Jobs
- Admin Panel for Job Management
- MVC Architecture

## 🛠️ Tools Used

- Spring Tool Suite (STS)
- Git & GitHub

## 📂 Project Structure

- Controller Layer
- Service Layer
- Repository Layer
- Thymeleaf Templates (Frontend)

## 🎯 Purpose

This project is developed as part of learning full-stack development using Java and Spring Boot.
